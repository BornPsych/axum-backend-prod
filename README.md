## Dev (REPL)

> NOTE: Install baocn with `cargo install bacon`.

```sh
# Terminal 1 - To run the server.
bacon run-long -- -q

# Terminal 2 - To run the quick_dev.
bacon ex -- -q --example quick_dev
```


## Dev

```sh
# Terminal 1 - To run the server.
cargo run

# Terminal 2 - To run the tests.
cargo run --example quick_dev
```
